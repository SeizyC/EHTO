# EHTO 개발계획

## 1. 개발 목표

단순 AI 채팅이 아니라, 사용자가 자신만의 작은 사회 안에 머무르는 경험을 만든다. MVP는 한국어 기반 개인 World 1개를 중심으로 다음 감각을 우선 구현한다.

- 사용자가 없어도 살아있는 듯한 월드 피드
- 멤버 간 관계 / 분위기 / 사건의 누적 변화
- 채팅창이 아닌 Ambient Feed
- World Identity Drift가 화면과 대화에 반영
- 반복 피로도를 줄이는 이벤트 / 리듬 / 화제 전환

## 2. MVP 범위

### 포함

- 사용자별 개인 World 자동 생성
- 멤버 12명 생성 / 5명 내외 활성화 (UI상 "AI" 단어 미노출)
- Habbo 풍 아이소메트릭 룸 렌더링 + 픽셀 미니미 (3/4 stance, 모듈식)
- World Feed (대화 / presence / 이벤트 / 미디어 / world drift)
- World State Manager
- Social Director Engine
- Relationship Mutation Engine
- Narrative Event Engine
- Ambient Rhythm System
- User Signal Tracker
- Social Signature Builder
- Repetition Detection Engine (기본형)
- World Identity 화면
- 한국어 대화 생성 파이프라인

### 제외 / 후순위

- 실제 타 유저 월드 탐험
- 유료 티켓 결제
- 광고 / 스폰서 월드
- 고도화된 미디어 검증 자동화
- 대규모 샤딩 및 멀티 리전 확장

## 3. 권장 기술 스택

### Frontend
- Next.js (App Router) + React + TypeScript
- Tailwind CSS
- Zustand
- Framer Motion
- WebSocket / SSE
- 픽셀 렌더링: Canvas 또는 CSS 스프라이트 (현재 sprite renderer 방향)

### Backend
- Node.js + TypeScript + Fastify
- PostgreSQL (Supabase)
- Redis
- BullMQ
- pgvector

### AI Layer
- OpenAI Responses API (GPT 계열)
- embedding memory layer
- lightweight routing models

## 4. 페이즈

### Phase 0 — Bootstrap (완료 기준)
- Next.js 프로젝트 셋업
- 디자인 문서 / 개발계획 작성

### Phase 1 — 정적 Spatial Room 프로토타입
- Atmosphere Header / Isometric Room / Ambient Feed / Composer (더미 데이터)
- 모바일 360 / 390px
- 픽셀 미니미 sprite renderer (3/4 stance, body / outfit / hair)

### Phase 2 — World / Member / Feed API
- DB 스키마 (World / Member / RelationshipEdge / WorldEvent / FeedItem)
- World Session Manager
- 한국어 대화 생성 파이프라인 (Context Builder → Persona)

### Phase 3 — Social Director & Drift
- Orchestration / Timing
- Relationship Mutation
- Narrative Event Engine
- World Identity Drift 누적

### Phase 4 — Repetition Detection / Ritual / Polish
- Repetition Detection
- Ritual Engine
- 사용자 피드백 기반 톤 조정

## 5. 가드레일 (전 페이즈 공통)

- UI에 "AI / 챗봇 / 봇" 단어 노출 금지 → `member` / `presence`
- Habbo 풍 인간형 미니미 유지 (creature 방향 채택하지 않음)
- 외모 비교 / 꾸미기 압박 유발 금지 — 커스터마이즈는 최소
- 채팅창이 아닌 Ambient Feed
- 유저는 세계의 주인공이 아니라 일부
- "오늘도 저 세계가 살아있다" 감각
